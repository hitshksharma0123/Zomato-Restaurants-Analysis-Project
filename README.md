# Zomato-Restaurants-Analysis-Project

Project Overview

This project involves building an end-to-end analytics dashboard using global Zomato restaurant data to support data-driven decisions for restaurant expansion, pricing strategy, and customer experience optimization.

## Key Objectives

Analyze restaurant performance across countries, cities, and time periods

Identify growth opportunities and low-competition markets for new restaurants

Understand how pricing, ratings, and delivery options influence customer feedback

## Work Performed

Designed an interactive Excel dashboard to visualize restaurant trends and key business metrics.

Cleaned and standardized raw data by handling missing and inconsistent values and enriching records using lookup functions.

Analyzed restaurant distribution by country and year to identify market saturation and expansion potential.

Evaluated the relationship between pricing, ratings, voter counts, online delivery, and table booking on customer satisfaction.

Applied conditional logic, logical functions, and array formulas to derive insights on affordability and service preferences.

Assessed competition intensity, cuisine performance, and rating distribution to recommend optimal countries, cities, and cuisines for expansion.

## Tools & Techniques

Excel (Pivot Tables, Lookups, Conditional Formatting, Array Formulas)

Data Cleaning & Transformation

Exploratory Data Analysis (EDA)

Dashboarding & Data Visualization

## Key Outcomes

Identified high-potential markets with lower competition for restaurant expansion

Highlighted cuisines and pricing strategies associated with higher customer ratings

Delivered actionable insights through clear visual dashboards to support strategic decision-making
